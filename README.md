#Ablib Acme Board Python library

A Python library for controlling the pins on:

* [WIFIG25](http://armdevs.com/WiFiG25.html)

##Installation

```
# git clone git://github.com/averrin/ablib.git
# cd ablib
~/ablib# python setup.py install
```
